Различия между задачами «дробный рюкзак» и «0-1 рюкзак»
Тип выбора предметов:
В задаче «дробный рюкзак» каждый предмет можно брать частично (например, половину предмета). Здесь разрешается деление вещей на части.
В задаче «0-1 рюкзак» предметы берутся целиком («либо весь предмет берем, либо вообще не берем»). Деление невозможно.
