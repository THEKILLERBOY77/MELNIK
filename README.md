Алгоритм Christofides (или 2-аппроксимация) для задачи коммивояжера (TSP)
Цель: Найти решение задачи TSP с гарантированной ошибкой не больше чем вдвое превышающей оптимальное значение (метрика расстояния удовлетворяет неравенству треугольника).

Основные шаги реализации:
Шаг 1: Найдем минимальное остовное дерево (Minimum Spanning Tree, MST). Это задача решается стандартным алгоритмом Прима или Краскала. Получаем связывающий подграф графа минимальной стоимости.Сложность построения минимального остова дерева — O(E log V), где E = количество рёбер, V = количество вершин.
Шаг 2: Удваиваем все рёбра полученного остова. Тем самым мы создаём новый мультиграф с четностью степеней всех вершин (так как каждая вершина была связана чётным числом рёбер).Временная сложность удваивания рёбер — O(V^2), поскольку каждый элемент проверяется дважды (каждая вершина соединена с каждой второй ровно два раза).
Шаг 3: Поиск Эйлерова цикла (Эйлерова пути) в получившемся мультиграфе. Для этого используем стандартный алгоритм поиска Эйлерова маршрута, начиная с произвольной вершины. Эйлеров цикл проходит через каждое ребро ровно один раз.Сложность нахождения Эйлерова цикла — O(V + E), где E — количество рёбер в новом мультиграфе, которое примерно равно O(V²).
Шаг 4: Преобразование Эйлерова цикла в гамильтонов путь путём исключения повторяющихся городов («шорткат»). Проходим циклический путь, игнорируя повторы городов.Время преобразования — O(V), так как проходится последовательность длиной порядка числа вершин.
Шаг 5: Оценка качества полученного результата относительно оптимальной нижней границы.

Анализ временной сложности:
Минимальное остовное дерево (MST) — O(E log V) ≈ O(V² log V), так как полный граф имеет порядка V² рёбер.
Дублирование рёбер — O(V²).
Нахождение Эйлерова цикла — O(V²).
Преобразование в гамильтонов путь — O(V).
Итого общая временная сложность алгоритма составляет O(V² log V).

Оптимальность:
Алгоритм гарантирует решение, длина которого не превышает двухкратной оптимальности исходной задачи благодаря свойствам метрики пространства (неравенство треугольника соблюдается). Этот метод является популярным эвристическим решением, особенно эффективным для больших размеров задачи, где точное решение требует экспоненциального времени.
